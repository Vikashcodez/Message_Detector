# Message_Detector

Message_Detector is a machine learning application designed for text classification with a built-in user authentication system. It leverages Python for backend functionality and CSS/HTML for the frontend.

## Features
- **Text Classification**: Utilizes machine learning models for accurate message classification.
- **User Authentication**: Secure login and registration functionality.
- **Modern Interface**: Built using CSS and HTML for a clean and responsive user experience.

## Prerequisites
Ensure the following are installed:
- Python 3.7 or higher
- pip (Python package manager)

## Installation

Follow these steps to set up and run the application:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Vikashcodez/Message_Detector.git
   cd Message_Detector
   ```

2. **Install dependencies**:
   Use the `requirements.txt` file to install all necessary Python libraries:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**:
   Start the application by executing the `app.py` file:
   ```bash
   python app.py
   ```

4. **Access the application**:
   Open your browser and navigate to `http://127.0.0.1:5000` to use the application.

## Project Structure
- **`app.py`**: The main entry point for the application.
- **`requirements.txt`**: Contains the list of dependencies required to run the application.
- **Frontend**: HTML and CSS files for the user interface.
- **Backend**: Python modules for text classification and authentication.
